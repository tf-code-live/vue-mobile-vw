### 项目说明
- 技术栈 vue2.0 + vant (按需加载)
- postcss-px-to-viewport 移动端适配
- eslint-loader 开发环境自动修复
- 可在vue.config.js中 的css > loaderOptions > less > [lessOptions] > modifyVars中定制vant的主题颜色
- 重置css样式 reset.css
